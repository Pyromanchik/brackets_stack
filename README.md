# Проверка сбалансированности скобок

Программа проверяет, является ли последовательность скобок сбалансированной. Для этого используется стек, реализованный на основе Python-списка.

## Стек (Stack)

Реализован абстрактный тип данных **стек** с методами:
- `is_empty()` — проверка на пустоту
- `push(item)` — добавление элемента на вершину
- `pop()` — удаление и возврат верхнего элемента
- `peek()` — просмотр верхнего элемента без удаления
- `size()` — количество элементов в стеке
